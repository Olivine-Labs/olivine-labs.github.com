Alchemy Websockets

An open-source C# Websocket server built to handle hundreds of thousands of simultaneous connections.

Check it out on GitHub >

We built Alchemy Websockets as a way to use the latest in web technology with a familiar and flexible development environment. C# allows high scalability through microthreading and provides an easy development platform. And then we wanted to share it with everyone, so we made it freely available on GitHub under the permissive LGPL license.

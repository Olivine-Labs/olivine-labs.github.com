Tiamat

A server architecture maintainance application.

Tiamat, still in production, is an application framework that handles request routing for applications and a management API for highly-scaled application servers. It handles HTTP and WebSockets by default, and uses a custom TCP connection protocol internally. It is due for public release late Q1 2012.
